# DMC Color Matcher Pro

プロフェッショナル刺繍糸カラーマッチングアプリケーション

## 機能

- 📷 画像アップロード＆ズーム
- 🎨 スポイトツールで色抽出
- 🧵 DMC刺繍糸への自動マッチング
- 📋 Color Legend（抽出した色の一覧）
- 📍 DMC色番号の収納位置表示（列-行）
- 📥 CSV/PDF出力

## 使い方

1. 画像をアップロード
2. スポイトツールで色をピックアップ
3. 「DMC色を検索」ボタンをクリック
4. 自動的にColor Legendに追加されます
